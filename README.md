En esta practica se realizó un crud de empleados en java con base de datos potgres usando TOMCAT 9, con el servicio teniendo que encenderse y apagarse de manera local dentro de VS Code y copiando el .war creado
a la respectiva carpeta de webapps de Tomcat. Además del anexo de un inicio de sesión sencillo previo al ingreso de vista de los datos.
